# TypeScript: Generated Loader Files

Generated by `bin/rails reactive_view:types:generate`.

## Locations

- `.reactive_view/types/loaders/<route>.ts`
- `.reactive_view/types/loader-data.d.ts`

## Per-route file content

Depending on loader definitions, a generated file can include:

- `LoaderData` interface
- `getLoaderData`, `preloadData`, `useLoaderData`
- mutation interfaces
- `<mutation>Action` constants
- `<Mutation>Form` components
- `useForm(name)` helper
- `useStream(name)` helper

When streams are present, `useForm(stream)` is also supported and returns a stream-bound `Form` component.

## Central map

`loader-data.d.ts` augments `@reactive-view/core` `LoaderDataMap` for cross-route typed usage.

## Regeneration triggers

- shape definition changes
- mutation/action assignment changes
- loader file add/remove/rename

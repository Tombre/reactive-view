# TypeScript: Core API (`@reactive-view/core`)

`@reactive-view/core` re-exports SolidJS and Solid Router APIs plus ReactiveView helpers.

## ReactiveView exports

- Loader: `useLoaderData`, `createLoaderQuery`
- Mutations: `createMutation`, `createJsonMutation`, `useAction`, `useSubmission`, `useSubmissions`
- Streaming: `createStream`
- CSRF: `getCSRFToken`, `getCSRFParam`
- Types: `LoaderDataMap`, `LoaderData`, `HasLoaderData`

## Related entry points

- `@reactive-view/core/vite-plugin`
- `@reactive-view/core/config`
